              2. 主要カード効果一覧
================================================================================

【サイバー・ドラゴン・コア】
├ 召喚時：「サイバー」魔法・罠をサーチ
└ フィールド・墓地で「サイバー・ドラゴン」扱い（レベル5）

【サイバーダーク・ワールド】（永続魔法）
├ ①発動時：墓地に同名がいない「サイバーダーク」をサーチ
├ ②メインフェイズに「サイバーダーク」の召喚を行う
└ ③サイバーダークの装備効果で相手墓地からも装備可能

【サイバー・ダーク・キメラ】
└ 墓地へ送られた場合：パワー・ボンドをサーチ

【サイバー・ダーク・クロー】
├ ①手札から捨てて「サイバーダーク」魔法・罠サーチ
├ ②装備モンスターが戦闘時、EXデッキから1体墓地へ
└ ③装備状態で墓地へ送られた場合、墓地のサイバーダークを手札に

【サイバー・ドラゴン・ヘルツ】
├ 墓地へ送られた場合：「サイバー・ドラゴン」をサーチ
└ フィールド・墓地で「サイバー・ドラゴン」扱い

【サイバー・ダーク・ヴルム】
├ ①フィールド・墓地で「サイバー・ドラゴン」扱い
├ ②手札・墓地にいて、場か墓地に他の機械族サイバーがいる場合、
│   手札・デッキからサイドラを墓地へ送ってSS
│   → その後「サイバー」か「サイバネティック」魔法罠を墓地から回収可能
└ ※②はデュエル中1度

【サイバー・ヨルムンガンド】
├ 通常召喚不可
├ ①【相手場にモンスターがいる場合】手札からSS
│   → デッキからサイドラをSSか装備
└ ②自分のサイドラを手札に戻して「融合」をサーチ
※先攻では①が使えない（相手場が空のため）

【機械仕掛けの騎士】（リンク1）
├ 素材：攻撃力1000以下の機械族1体
├ ①L召喚時、永続魔法を墓地へ送り「機械仕掛けの夜」をサーチ
└ ②墓地の攻撃力1000以下機械を、他の機械をリリースしてSS

【セリオンズ"キング"レギュラス】
├ ①墓地の「セリオンズ」か機械族を対象に、手札からSS＆装備
└ ②相手の効果発動時、手札・場の「セリオンズ」を墓地へ送って無効
※自身を墓地に送っても無効にできる

【サイバー・ドラゴン・ノヴァ】（ランク5エクシーズ）
├ 素材：機械族レベル5×2
├ ①素材1つ取り除き、墓地のサイドラをSS
├ ②手札・場のサイドラを除外して攻撃力+2100
└ ③相手効果で墓地へ送られた場合、EXから機械族融合をSS

【サイバー・ドラゴン・インフィニティ】（ランク6エクシーズ）
├ ノヴァの上に重ねてX召喚可能
├ ①素材の数×200アップ
├ ②1ターンに1度、表側攻撃表示モンスターを素材にする
└ ③1ターンに1度、素材1つ取り除き、効果発動を無効にして破壊

【サイバー・ドラゴン・ズィーガー】（リンク2）
├ 素材：機械族モンスター2体
├ ①自分の機械族の攻撃力を倍にする（※直接攻撃不可になる）
└ ②戦闘で破壊されない

【黎銘機ヘオスヴァローグ】（リンク1）★重要★
├ 素材：機械族・光属性モンスター1体
├ ①融合召喚した場合、次ターンのスタンバイに融合魔法を墓地から回収
└ ②相手が魔法・罠の効果を発動した時、
    場の機械族・光属性1体 or 墓地の機械族・光属性2体を除外して
    → その発動を無効にする
※②は魔法・罠専用の妨害。モンスター効果は止められない
※素材になれるのは：コア（光）、ヘルツ（光）のみ
  キメラ（闇）、ヴルム（闇）、騎士（地）は不可

【vs 巳剣（Tier1・14%）】

┌───────────────────────────────────────────────────────────────┐
│ 特徴                                                         │
├───────────────────────────────────────────────────────────────┤
│ ・儀式モンスターによる妨害                                   │
│ ・サーチを多用（ドロール有効）                               │
│ ・EXゾーンの使用は少なめ                                     │
└───────────────────────────────────────────────────────────────┘

【対策】
┌─────────┬────────────────────────────────────┐
│ 最優先   │ 壊獣 → 儀式モンスターをリリース除去│
│ 次点     │ 一滴 → 複数妨害を無効に           │
│ 有効     │ メガフリートは使いにくい（EXゾーン不使用）│
└─────────┴────────────────────────────────────┘

【ワンキルルート】
壊獣を相手に送りつけ → パワボエンド（8000）+ リミ解（16000）
→ 壊獣（3300）に攻撃 → 12700ダメージ → ワンキル

【vs ドラゴンテイル（Tier1・11%）】

┌───────────────────────────────────────────────────────────────┐
│ 特徴                                                         │
├───────────────────────────────────────────────────────────────┤
│ ・烙印融合からアルバス融合体展開                             │
│ ・ミラジェイド：対象を取らない除外 + ターン終了時全破壊       │
│ ・ドラゴン族が多い                                           │
└───────────────────────────────────────────────────────────────┘

【対策】
┌─────────┬────────────────────────────────────┐
│ 最優先   │ 超融合 or 壊獣 → ミラジェイドを処理│
│ 次点     │ 一滴 → ミラジェイドを無効化       │
│ 注意     │ ミラジェイドは場を離れさせないとターン終了時効果が発動│
└─────────┴────────────────────────────────────┘

【ミラジェイド対面の正しい動き】
・超融合で素材に → ターン終了時効果発動しない
・壊獣でリリース → ターン終了時効果発動しない

- クローと一緒に引いている → クローから動いてワールドを温存
- コアと一緒に引いている → コアでワールドサーチは不要
